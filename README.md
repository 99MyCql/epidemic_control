# epidemic_control

疫情防控小程序后端

## Getting Start

环境：

- python >= 3.7
- MySQL

代码规范：[PEP8](https://peps.python.org/pep-0008)

安装依赖（建议使用虚拟环境）：

```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

修改配置文件：`./config.yaml`

运行:

```bash
python app.py
```
